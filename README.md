# sprint-7

## Descripción del proyecto 
El aplicativo permite ver a detalle la información de los anuncios de venta de vehiculos  

- En la aplicación se puede encontrar un histograma de la distribución de los odómetros de los vehiculos 
- Gráfico de dispersión donde se entre la relación entre precio y kilometraje de cada vehiculo
- Botones dinámicos para generar visualizaciones del histograma y el diagrama de dispersión

## Aplicativo web implementado en streamlit    
- Pandas 
- Plotly
